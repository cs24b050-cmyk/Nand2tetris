# Nand2Tetris

Implementation of the Nand2Tetris course projects, covering the complete construction of a computer system from logic gates to a software toolchain capable of executing high-level programs.

# Intro

This repository contains my solutions to the Nand2Tetris course projects. The objective of the course is to build a complete 16-bit computer system from first principles and develop the corresponding software stack required to run programs on the custom hardware platform.

# Hardware Components

Implemented using HDL:

-> Logic Gates
-> Combinational Circuits
-> Sequential Circuits
-> Registers
-> Arithmetic Logic Unit (ALU)
-> CPU
-> Memory Hierarchy
-> Hack Computer Architecture

# Software Components

Implemented in C++:

-> Assembler
-> Virtual Machine (VM) Translator
-> Jack Compiler
-> Operating System Modules

# Key Concepts

-> Digital Logic Design
-> Computer Architecture
-> Assembly Language
-> Virtual Machines
-> Compiler Construction
-> Systems Programming
